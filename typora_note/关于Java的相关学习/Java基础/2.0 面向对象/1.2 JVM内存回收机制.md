源代码->字节码文件(编译生成)->解释运行

内存解析，是在运行的时候，编译完是个文件（字节码文件）。



编译完源程序后，生成一个或者多个字节码文件。

我们使用JVM中类的加载器和解释器对生成的字节码文件进行解释运行。意味着，需要将字节码文件对应的类加载到内存中，涉及到内存解析。
